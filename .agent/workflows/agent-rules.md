# Agent Behavior Rules

## Verification and Screenshots
> [!IMPORTANT]
> **CRITICAL RULE**: Do not take screenshots after making changes (e.g., to save in the `walkthrough.md` document or any other artifact) unless explicitly asked by the user to do so.
> **CRITICAL RULE**: Do not perform any verification steps (testing, checking UI in the browser, etc.) unless explicitly asked by the user. The user will handle all verification.
